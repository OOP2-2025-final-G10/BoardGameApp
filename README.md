# BoardGameApp
BoardGameApp は、Flask を用いて構築された基本的な双六（すごろく）ゲームアプリケーションです。
複数人のユーザーが同時に参加し、順番にサイコロ（ルーレット）を回してマスを進み、
イベントや所持金の増減、株式取引などを楽しむことができます。

## 特徴
ブラウザ上でプレイできる双六ゲーム

複数ユーザーによる同時参加型プレイ

株価売買による所持金の増減

## 使用技術

Python

Flask

HTML / CSS / JavaScript

Chart.js（株価グラフ表示）
